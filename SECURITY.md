# Security Policy

## Reporting a Vulnerability

We take security vulnerabilities seriously. If you discover a security vulnerability in Njord, please report it responsibly.

**Please do not open a public issue for security vulnerabilities.**

Instead, email your report to: **security@njord-app.dev**

Include the following information in your report:

- Description of the vulnerability
- Steps to reproduce (if applicable)
- Potential impact
- Suggested remediation (if available)

We will acknowledge your report promptly and work with you to address the vulnerability in a timely manner.
